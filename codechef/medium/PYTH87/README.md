## Problem

### Changing the elements of a List

Listen

You can also modify elements in a list by assigning a new value to a specific index.

```
fruits = ["apple", "banana", "orange", "grape"]
fruits[1] = "kiwi"
print(fruits)  # Output: ["apple", "kiwi", "orange", "grape"]

```

### Task

Write a program which does the following

- The 3rd month in the given list is incorrect
- Update the 3rd month in the given list with the correct one - "Mar"
- Once the 3rd list element is updated, output the entire list to the console

## Solution

**Language:** Python  
**Runtime:** N/A  
**Memory:** N/A  
**Submitted:** 2026-07-16T04:11:52.331Z  

```py
# Update the 3rd month in the list below to "Mar"

months = ["Jan", "Feb", "Dec", "Apr"]
months[2] = "Mar"
print(months)
```
