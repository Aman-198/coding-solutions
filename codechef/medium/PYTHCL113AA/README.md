## Problem

### Range of numbers

Listen

Write a program to print integers from 30 to 5 (including 5) with a decrement value of 1

## Solution

**Language:** Python  
**Runtime:** N/A  
**Memory:** N/A  
**Submitted:** 2026-07-19T08:31:45.628Z  

```py
# Update the '_' to solve the problem
for i in range(30, 4, -1):
    print(i)
```
